Why Dark Mode?

Dark mode reduces the light emitted by device screens while maintaining the minimum color contrast ratios required for readability.

  • Reduces eye strain
  • Saves battery life (OLED screens)
  • Looks aesthetically pleasing

How it Works

We use JavaScript to toggle a specific class (.dark-mode) on the <body> element. CSS Variables update automatically when this class is applied.

We also use LocalStorage to remember your preference when you refresh the page!
