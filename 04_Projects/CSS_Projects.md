# 4.2 CSS Projects

## Goal
Make it look good! Focus on **Layouts**, **Colors**, and **Responsiveness**.

---

## Project 1: Personal Portfolio
**Objective**: A personal website to showcase your work.

### Requirements
1.  **Navbar**: Fixed at the top, links to About, Work, Contact.
2.  **Hero Section**: Full-screen background image/color with your name centered.
3.  **Projects Grid**: Using **CSS Grid** to display project cards.
4.  **Contact Section**: Styled form.
5.  **Responsive**: Must look good on Mobile (stack columns) and Desktop.

### Key Concepts Used
- Flexbox (Navbar)
- Grid (Project Gallery)
- Position (Sticky Header)
- Media Queries (Responsiveness)

---

## Project 2: Product Landing Page
**Objective**: A sales page for a product (e.g., Headphones).

### Requirements
1.  **Header**: Logo and Nav.
2.  **Video**: Embed a YouTube video explaining the product.
3.  **Features**: 3-column layout using Flexbox.
4.  **Pricing**: Pricing table comparing 3 plans.
5.  **Call to Action (CTA)**: Big button "Buy Now".

### Key Concepts Used
- Flexbox alignment (`justify-content`, `align-items`)
- Embedded Media
- CSS Buttons (Hover effects, Transitions)

---

## Project 3: Parallax Website
**Objective**: A storytelling site with scrolling effects.

### Requirements
1.  **Backgrounds**: Multiple sections with `background-attachment: fixed`.
2.  **Typography**: Large, readable fonts.
3.  **Transparency**: Text boxes with semi-transparent backgrounds over images.

### Key Concepts Used
- Background Properties (`cover`, `fixed`)
- RGBA Colors
- `z-index` layering
