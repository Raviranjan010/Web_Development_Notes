# 4.1 HTML Only Projects

## Goal
Build these projects using **pure HTML**. No CSS (or very minimal inline styles if strictly needed for layout), no JavaScript. Focus entirely on **Structure** and **Semantics**.

---

## Project 1: Tribute Page
**Objective**: Create a page dedicated to a famous person (real or fictional).

### Requirements
1.  **Image**: Display a photo of the subject.
2.  **Title**: Who is this person? (`<h1>`).
3.  **Timeline**: Use a List (`<ul>`) to show key events in their life.
4.  **Quote**: Use `<blockquote>` for a famous quote.
5.  **Link**: A link to their Wikipedia page (`target="_blank"`).

### Key Concepts Used
- Images (`<img>`)
- Lists (`<ul>`, `<li>`)
- Text Formatting (`<strong>`, `<em>`)
- Links (`<a>`)

---

## Project 2: Survey Form
**Objective**: Create a form to collect user data for a fictional event.

### Requirements
1.  **Inputs**: Name, Email, Age.
2.  **Dropdown**: "Which option describes you best?" (`<select>`).
3.  **Radio Buttons**: "Would you recommend us?" (Yes/No).
4.  **Checkboxes**: "What would you like to see improved?" (Multiple options).
5.  **Textarea**: "Any comments?".
6.  **Submit Button**.

### Key Concepts Used
- Forms (`<form>`)
- Input Types (`text`, `email`, `number`, `radio`, `checkbox`)
- Labels (`<label for="...">`)

---

## Project 3: Technical Documentation Page
**Objective**: Documentation for a coding topic (e.g., "Python Basics").

### Requirements
1.  **Sidebar Navigation**: Links that jump to sections on the same page (use Anchor Links `#section-id`).
2.  **Code Blocks**: Use `<pre><code>` to display code snippets.
3.  **Sections**: Multiple `<section>` elements with `<header>` titles.

### Key Concepts Used
- Semantic HTML (`<nav>`, `<main>`, `<section>`, `<header>`)
- Internal Linking (`href="#topic"`)
- Preformatted Text (`<pre>`)
