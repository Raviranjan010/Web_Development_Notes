[ HTML QUICK REVISION CHEAT SHEET ]

1. STRUCTURE
   <!DOCTYPE html>          -> Defines HTML5 document
   <html>...</html>         -> Root element
   <head>...</head>         -> Metadata, title, links (invisible)
   <body>...</body>         -> Visible content

2. TEXT TAGS
   <h1> to <h6>             -> Headings (h1 = most important)
   <p>                      -> Paragraph
   <br>                     -> Line Break (Self-closing)
   <hr>                     -> Horizontal Rule (Self-closing)
   <strong> / <b>           -> Bold
   <em> / <i>               -> Italic

3. LINKS & IMAGES
   <a href="url">Text</a>   -> Link (Anchor)
   <img src="url" alt="">   -> Image (Self-closing)

4. LISTS
   <ul> <li>...</li> </ul>  -> Unordered List (Bullets)
   <ol> <li>...</li> </ol>  -> Ordered List (Numbers)

5. TABLES
   <table>                  -> Container
     <tr>                   -> Table Row
       <th>Header</th>      -> Header Cell (Bold)
       <td>Data</td>        -> Standard Cell
     </tr>
   </table>

6. FORMS
   <form action="" method="">
     <input type="text">    -> One-line text
     <input type="password">-> Hidden text
     <button>Submit</button>
   </form>

7. SEMANTIC TAGS (Use these instead of divs for layout)
   <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>

8. PRO TIPS
   - Always include 'alt' attribute on images.
   - Only ONE <h1> per page.
   - Use 'id' for unique elements, 'class' for multiple elements.
