[ CSS QUICK REVISION CHEAT SHEET ]

1. SELECTORS
   element    { ... }       -> Selects all <element> (e.g., p {})
   .class     { ... }       -> Selects class="class"
   #id        { ... }       -> Selects id="id"
   *          { ... }       -> Universal selector (All)
   div p      { ... }       -> Descendant (p inside div)
   div > p    { ... }       -> Direct Child

2. BOX MODEL (Crucial)
   width / height           -> Content size
   padding                  -> Space INSIDE box
   border                   -> Line around padding
   margin                   -> Space OUTSIDE box
   * { box-sizing: border-box; } -> Makes sizing intuitive!

3. DISPLAY
   block                    -> New line, full width (div, p, h1)
   inline                   -> Same line, no width/height (span, a)
   inline-block             -> Same line, respects width/height
   none                     -> Hides element completely

4. FLEXBOX (1D Layout)
   container { display: flex; }
   justify-content: center; -> Horizontal Alignment
   align-items: center;     -> Vertical Alignment
   flex-direction: column;  -> Stack items

5. GRID (2D Layout)
   container { display: grid; }
   grid-template-columns: 1fr 1fr; -> Two equal columns
   gap: 20px;               -> Space between items

6. POSITION
   static                   -> Default
   relative                 -> Normal position, but can use top/left
   absolute                 -> Removed from flow, relative to nearest positioned parent
   fixed                    -> Relative to browser window (sticky header)

7. UNITS
   px                       -> Absolute (Fixed)
   rem                      -> Relative to Root font-size (Best for Text)
   %                        -> Relative to Parent
   vh / vw                  -> Viewport Height/Width

8. COLORS
   #FF0000                  -> Hex
   rgb(255, 0, 0)           -> RGB
   rgba(255, 0, 0, 0.5)     -> RGB + Opacity (0.5)
